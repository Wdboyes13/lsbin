# lsbin
Just a tool to list libraries linked to an executable.  

## Usage
```
lsbin [OPTIONS] <FILE...>

FILE        The files(s) to process
Options:
    --help, -h      Show this text and exit
    --json, -j      Output JSON data
    --text, -t      Output plaintext data, default
    --license, -l   Print license and copyright
```
